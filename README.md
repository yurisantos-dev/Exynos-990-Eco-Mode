# Exynos-990-Eco-Mode
KernelSU module that toggles the CPU clock based on Android Power Saving mode, below android default values.

## Clusters values

- big: 1.69GHz (Stock: 2.730GHz)
- Middle: 1.508GHz (Stock: 2.504GHz)
- LITTLE: 1.3GHz (Stock: 2.002GHz)

The values ​​can be adjusted on lines 37, 38, and 39 of the "service.sh" file, in accordance with the CPU frequency table.
